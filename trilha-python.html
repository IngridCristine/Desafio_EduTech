Trilha de Programação em Python

 Este curso é outra alternativa para a 3ª Série do Ensino Médio e do Técnico, e nele vamos estudar a linguagem de programação mais querida dos desenvolvedores na atualidade. Python é uma das maiores escolhas para entrar no mundo da programação, onde com ela poderemos desenvolver desde os conceitos mais básicos da programação evoluindo até a possibilidade do desenvolvimento de jogos com essa poderosa linguagem de programação.

 Com Python, atualmente podemos trabalhar desde aplicações que manipulam informações simples, como dados de hardware, até informações complexas de alto poder de processamento, como ações dentro de games. Python é uma linguagem em ampla e constante evolução e adesão para aplicações de pequeno, médio e grande porte.
